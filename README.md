## GETTING STARTED

### Clone The Project

```
$ git clone https://github.com/placideirandora/todo-list-app-with-react.js
```

### Install Required Dependencies

```
$ npm install
```

### Start The Development Server

```
$ npm run dev
```

### Start The Production Server

```
$ npm run build
```

```
$ npm start
```

```
$ Visit http://localhost:8080 with your browser
```
